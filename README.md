## Overview
LogiQ AI is your smart ai assistant with the the power to translate words, generate images, code, hack, uncensored chat bot, and much more!

### Key features
- Real-time chat UI
- Weather, time & date, and battery viewer widgets.
- Fuctions
- Different chatbots
- Focus mode (to focus on the chat)
- Image generation
- Accent color changing
- Settings
- Customization
- And a bunch of more features!

### Tech stack
- Frontend: Next.js / React (adjust if you used something else)
- Backend: Node.js / Express / FastAPI (replace accordingly)
- Datastore: PostgreSQL (or your DB)
- Auth: Auth0 / NextAuth / Custom JWT
- Hosting: Base44 / Vercel / Netlify / Docker
