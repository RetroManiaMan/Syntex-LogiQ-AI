## How to contribute
1. Fork the repo.
2. Create a feature branch: `git checkout -b feat/my-feature`
3. Keep changes focused and add tests.
4. Open a pull request to `main`.

## Code style
- Use Prettier and ESLint (run `npm run lint`).

## PR requirements
- Description of changes
- Tests or screenshots where applicable
- Link to the issue (if any)
